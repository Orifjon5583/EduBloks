# EduBlocks

Dasturlash ta'limiga mo'ljallangan zamonaviy veb-platforma.

## Tech Stack
- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **Database:** PostgreSQL + Prisma ORM
- **Real-time:** Socket.io

## Ishga tushirish

```bash
# Server
cd server
npm install
npx prisma migrate dev
npx prisma db seed
npm run dev

# Client
cd client
npm install
npm run dev
```
